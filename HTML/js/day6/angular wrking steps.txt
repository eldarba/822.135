שלבים בעבודה עם אנגולר
1. הורדנו node.js - סביבת ריצה של JS
2. השתמשנו ב npm של node.js כדי להתקין Angular
3. יצרנו תיקיית workspace
===============================
4. השתמשנו בפקודה ng new כדי ליצור פרויקט חדש ng new my-app
5. קיבלנו תיקייה בשם my-app עם המון קבצים בפנים.
6. פתחנו CMD מתוך התיקייה my-app והפעלנו את הפקודה ng s -o
על מנת להרים את השרת ng s ולפתוח דפדפן o-
7. עולה השרת ונפתח הדפדפן ואנחנו רואים את הדף הראשי

מחקנו את הקומפוננט הראשי שהיה מוגדר
במקומו יצרנו 2 קומפוננטים חדשים
הצבנו את root component על index html
ראינו את התוכן של הקומפוננט

ng g c components/root --skip-tests --dry-run
